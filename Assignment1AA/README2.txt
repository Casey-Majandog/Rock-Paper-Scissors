Server GUI
How to open and use server GUI:
1. Go into doc -> Server_GUI.jar
2. Open up the command prompt
3. Type in java -jar
4. Drag the Server_GUI.jar into the command prompt
5. Make sure there's a space between -jar and the file EG. 
User>java -jar [FILE]
6. Press enter and press "Start server" to put the server up
